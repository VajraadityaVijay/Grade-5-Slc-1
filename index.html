Multiplication
Steps to do multiplication on paper:
  • First, write the larger number above the smaller number.
  • Next, multiply the ones and then the tens and so on.
  • Finally, check your answers using a calculator.
Steps to do long multiplication on paper:
  • First, write the larger number above the smaller number.
  • Next, multiply the number in the ones place of the bottom number by the number in the ones place of the top number, then the tens place of the top number, and so on.
  • Then, draw a 0 in the ones column below the first product and multiply the number in the tens place of the bottom number by the number in ones place of the top number, then the tens place of the top number, and so on.
  • Next, Add the digits of both products.
  • First, write the larger number above the smaller number.
